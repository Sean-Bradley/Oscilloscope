X Y Oscilloscope by seanwasere ytbe

The X Y Oscilloscope uses Stereo Input.

Left Input channel affects X axis and right input channel affects Y axis.

Playing sine wave in left channel and cosine wave in right channel draws a circle.

Works Best with Windows 10 and Edge browser (it's true).

  • Open 'Privacy' Settings from the Cortana input.
  • Allow Microsoft Edge to access Microphone
  • Open 'Sound' settings from the Cortana input box.
  • Manage 'Recording Devices' to select desired audio input, stero input preferred.
  • Refresh Oscilloscope Page in Edge Browser, Press 'START' button and allow website to access your microphone.
  • Make some noise
